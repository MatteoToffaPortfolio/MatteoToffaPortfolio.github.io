# Progetti

## Automazione DDT — Process Automation
- **Descrizione:** Pipeline Python per il parsing automatico di Documenti di Trasporto (DDT) in PDF. Estrae dati strutturati, confronta DDT con ordini, rileva incongruenze e genera un output Excel pronto per l'analisi. Elimina l'inserimento manuale azzerando gli errori di trascrizione.
- **Tecnologie:** Python, Pandas, PDF Parsing, Excel, Process Automation, Data Validation
- **Link:** https://MatteoToffaPortfolio.github.io/ddt-automation.html

## Retail Analysis Dashboard — Fidelity
- **Descrizione:** Analisi del programma fidelity su dataset retail reale. Segmentazione clienti tramite modello RFM, identificazione pattern d'acquisto e dashboard KPI in Power BI per supportare campagne di retention e decisioni commerciali.
- **Tecnologie:** Python, Power BI, RFM Analysis, Customer Segmentation, DAX
- **Link:** https://MatteoToffaPortfolio.github.io/fidelity-analysis.html

## Analisi Vendite & Giacenza — NonSoloSport
- **Descrizione:** Modello Excel professionale per l'analisi di vendite e giacenza taglie di una catena retail sportiva multistore (Verona, Padova, Vicenza). Dataset sintetico su 1.800 transazioni, 12 prodotti, 7 marchi, fatturato simulato €338.408. Include pivot filtrabili, sell-through analysis con semaforo verde/arancio/rosso e dashboard KPI. Generato interamente via script Python con openpyxl.
- **Tecnologie:** Excel, Python, openpyxl, Pandas, Pivot Table, Sell-Through Analysis, Retail Analytics
- **Link:** https://MatteoToffaPortfolio.github.io/excel-nonsolosport.html
